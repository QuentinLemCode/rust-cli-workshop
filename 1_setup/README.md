# Environnement Setup

Make your Rust programming ready


## Step 1 - Setup 

### Create a new application

```bash
cargo new crabby
```

### How to build


Launch `cargo`:

```bash
cargo build
```

### How to run


Launch `cargo`:

```bash
cargo run 
```

### How to test


Launch `cargo`:

```bash
cargo test 
```
